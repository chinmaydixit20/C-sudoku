SUDOKU

Features:

1) Can generate a random 9 x 9 puzzle for the user to solve.
2) Can open an existing saved .txt file containing the sudoku data(must be of .txt form).
3) Can save an existing game of sudoku into desired file of txt form.
4) Can AutuSolve a user entered puzzle and if mistakes are detected, throws errors.
5) Can provide hints to the user currently solving a puzzle.
6) Can check a completed puzzle.

Basic Working:

Works using the recursive backtracking algorithm. For user interface, GTK has been used.
Saves and resumes files using raw i/o functions. Uses makefile for execution.
